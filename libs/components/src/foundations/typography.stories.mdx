## Usage

In order to use the font `Inter`, we need to import the css files containing the fonts
to the app we want it to be rendered.
Example: `main.tsx` on our main app (`main-app`)

> The fonts are being installed using `@fontsource/inter` [Fontsource](https://github.com/fontsource/fontsource)

```js
import '@canvass/foundations/fonts';
```

To use the typography on our project, we can use the
[Text](?path=/docs/text--primary) component,
[Heading](?path=/docs/heading--primary) component
and any text on the page will be shown with this font family.

## Overriding Chakra Font

`foundations/fonts/index.ts`

```js
const fonts = {
  heading: `'Inter', sans-serif`,
  body: `'Inter', sans-serif`,
};
```

`theme/index.ts`

```js
import { extendTheme } from '@chakra-ui/react';
import { fonts } from '@canvass/foundations';

const theme = extendTheme({ fonts });
```
