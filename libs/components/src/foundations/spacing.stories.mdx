# Spacing

Just like your color scale, working from a defined spacing system allows you to work faster and more consistently.
Consistent and scalable spacing helps you eliminate guesswork whilst designing and developing because you're designing with a limited set of options.

## Chakra UI

The space key allows you to customize the global spacing and sizing scale for your project.
By default these spacing value can be referenced by the `padding`, `margin`, and `top`, `left`, `right`, `bottom` styles.

### Overriding Chakra Spacing

```js
import { extendTheme } from '@chakra-ui/react';

const spacing = {
  space: customSpacing,
};

const theme = extendTheme({ ...spacing });
```

### Default Spacing

<Canvas>
  <Story name="spacing" args={{ spaces: space }}>
    {Template.bind({})}
  </Story>
</Canvas>

By default, Chakra includes a comprehensive numeric spacing scale inspired by Tailwind CSS.
The values are proportional, so 1 spacing unit is equal to `0.25rem`, which translates to `4px` by default in common browsers.

> Mental model: If you need a spacing of 40px, divide it by 4. That'll give you 10. Then use it in your component.

### Reference

Refer to the [Chakra UI docs](https://chakra-ui.com/docs/styled-system/theme#spacing) for a list of spacing tokens and usage with style props.
